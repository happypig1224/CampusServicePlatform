绥E站是一款校园生活服务项目，为学生提供便捷的校园服务
技术选型
SpringBoot3.x + Mybatis + Mybatis-plus + Thymeleaf + MySQL8.x
核心模块：
1. 用户模块
- 用户注册/登录
    -  邮箱/手机号验证注册(先不做，显示当前功能正在开发)
    - 登录支持 验证码登录 / 第三方快捷登录(先不做，显示当前功能正在开发)
    - 通过用户名、密码登录
        - 密码加密处理
    - 注册通过用户名、二次密码校验注册
- 个人信息管理
    - 修改昵称、头像、密码等
- 角色区分(内置超级管理员账号)
    - 超级管理员可以管理用户和管理员
    - 其他功能管理员和超级管理员一样都是对资源进行管理
2. 校园论坛/社交圈子
- 版块划分：如“学习交流”、“社团天地”、“情感树洞”等
- 发帖与回复：用户自由发帖、回复、点赞、收藏
    - 用户发帖/回帖频率限制（防刷屏）
    - 不能重复点赞和收藏
- 支持关键字和按板块搜索
3. 二手交易市场(模仿闲鱼)
- 商品发布和编辑
    - 用户可以发布闲置物品(书籍、电子产品、生活用品等)，包含图片、描述、价格，联系方式
        - 图片继承腾讯云
    - 商品状态管理：上架中、已售出、下架、审核中
    - 商品浏览和搜索:按分类、关键词搜索商品
    - 商品收藏与评论:用户可以对感兴趣的商品进行收藏和留言
    - 交易沟通:集成站内通信，包含聊天功能
    - 下单功能
        - 买家：下单(包含地址篇、联系号码)、支付(先模拟支付成功)、待接单、待派送、派送中、已完成、确认收货
        - 卖家：全部、发单商品管理、接单、派送、完成
        - 订单超时自动关闭(设置10分钟)
        - 支付模拟完成后，生成虚拟支付凭证编号
4. 失物招领
- 寻物启事：用户发布丢失物品的信息（时间、地点、物品特征、图片）、
    - 可添加“紧急程度”标签（普通 / 紧急），帮助优先处理重要物品
- 拾物招领：用户捡到物品后，发布招领信息
- 信息匹配与认领：系统可尝试关键词匹配，失主可在线联系认领使(用前面的内置聊天功能)
- 感谢信功能：成功寻回物品后，可以发布公开动态展示在社区首页，营造正能量氛围
5. 校园跑腿/代办服务
- 任务发布：学生可以发布代取快递、代买零食、代打印等任务，并设置酬金
    - 酬金设置范围：必须大于3元，防止滥用
- 任务接单：其他有空闲时间的同学可以接单
    - 任务流程管理：包含“待接单-进行中-已完成-已支付”等状态，以及评价体系
    - 跑腿者可上传完成证明（照片、签收截图等）
- 酬金流水
    - 增加保险机制或平台担保：酬金暂存平台，确认完成后再打款给跑腿者
6. 资源共享
- 笔记共享：上传PDF/PPT/手写笔记，按课程分类
- 作业求助区：提问+解答
- 考试资料库：历年真题、复习提纲整理
- 组队学习：发起自习小组、考研搭子匹配
