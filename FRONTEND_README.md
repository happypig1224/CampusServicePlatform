# 绥E站校园生活服务平台 - 前端项目说明

## 项目概述

绥E站校园生活服务平台是一个面向高校学生的综合性服务平台，旨在提供便捷、高效的校园生活服务。平台包含用户管理、校园论坛、二手市场、失物招领、校园跑腿和资源共享等核心功能模块。

## 技术栈

- **前端框架**: HTML5 + CSS3 + JavaScript (ES6+)
- **UI框架**: Bootstrap 5.3
- **模板引擎**: Thymeleaf
- **图标库**: Font Awesome 6
- **AJAX请求**: Fetch API
- **响应式设计**: 移动优先，适配各种设备

## 项目结构

```
src/main/resources/
├── static/

│   ├── js/

```

## 部署说明

1. 确保后端服务正常运行
2. 将前端资源文件部署到Web服务器
3. 配置API基础URL
4. 测试各项功能是否正常

## 注意事项

1. 所有API请求需要携带认证Token
2. 文件上传有大小限制
3. 敏感操作需要二次确认
4. 表单提交前需要进行验证
5. 页面加载时需要显示加载状态
